# web concept

`index.html` をブラウザで開くと、文字落としゲームだけを残した Web 版デモが動きます。

- 背景とゲーム: 文字落としゲーム風アニメーション
- キーボード操作: `A` 左、`S` 落下、`D` 右、`W` 回転
- スマホ操作: 画面下の `回転 / 左 / 右 / 落下` ボタン
- リセット: `R` またはクリック / タップ
- 役: 指定された文字列が横または縦で揃うと消去
- 得点: 5文字 200点、4文字 100点、3文字 50点、`マチ` は 10点、末尾 `♥` で2倍
- 連鎖: 消去後の落下で続けて揃うと 2x / 4x / 8x ...
- 禁止語: `マンコ` が揃うとゲームオーバーで 0 点

簡易サーバーで開く場合:

```bash
cd /Users/sjdjeb3gmail.com/Downloads/photo/photo/web
python3 -m http.server 4173
```

その後、`http://localhost:4173` をブラウザで開いてください。
